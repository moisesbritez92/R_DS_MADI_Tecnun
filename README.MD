# Data Science Repository: R Lake

Este repositorio es un **Lake de R**, un espacio centralizado que recopila y organiza recursos, scripts, datos y ejercicios relacionados con el aprendizaje y la práctica de **Data Science** utilizando el lenguaje de programación **R**.

## ¿Qué encontrarás en este repositorio?

1. **Scripts y Ejercicios**:
   - Ejercicios prácticos organizados por días y temas.
   - Scripts que cubren desde manipulación de datos hasta modelado estadístico avanzado.

2. **Datos**:
   - Conjuntos de datos en formatos como `.csv`, `.json`, entre otros, utilizados para análisis y visualización.

3. **Chuletas y Guías**:
   - Documentos de referencia rápida para comandos y flujos de trabajo en R.

4. **Proyectos y Exámenes**:
   - Soluciones a ejercicios de exámenes y proyectos prácticos.

## Estructura del repositorio

- **`Dia_1_JF/`**: Introducción a R y manipulación básica de datos.
- **`Dia_2_JF/`**: Análisis exploratorio y estadística descriptiva.
- **`Dia_3_JF/`**: Visualización avanzada y análisis multivariante.
- **`Dia_4_JF/`**: Modelado estadístico y machine learning.
- **`Chuleta/`**: Chuletas y guías rápidas para R.
- **`data/`**: Conjuntos de datos utilizados en los ejercicios.

## ¿Qué es un Lake de R?

Un **Lake de R** es un repositorio que actúa como un lago de datos y recursos, donde se centralizan herramientas, scripts y datos para facilitar el aprendizaje y la práctica de R. Este enfoque permite a los usuarios acceder rápidamente a ejemplos prácticos y reutilizar código para resolver problemas de Data Science.

## ¿Cómo usar este repositorio?

1. **Explora los directorios**: Cada carpeta contiene ejercicios y datos organizados por temas.
2. **Ejecuta los scripts**: Abre los scripts en tu IDE favorito (como RStudio o Visual Studio Code) y sigue las instrucciones.
3. **Consulta las chuletas**: Usa las guías rápidas para recordar comandos y flujos de trabajo.
